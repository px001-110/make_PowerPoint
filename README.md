複数のWordファイルから、PowerPointを自動生成するためのwebアプリ。

1行目をタイトル,2行目以降は段落ごとにスライドを作成していく。
段落の1行目をスライドのタイトル、2行目以降を本文にしていく。

# 開発背景
Wordファイルを原稿としてPowerPointを作成する業務がほぼ毎週発生していた。
コピーアンドペーストでできる作業だったが、量が多くなるとミスも発生していた。
パソコン操作に慣れていない人などは長時間かかっていた。
そこでこの作業を効率化できないかと考え、開発した。
